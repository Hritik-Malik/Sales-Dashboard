# Sales Performance Dashboard
# Project Summary

Built an interactive Sales Performance Dashboard to analyze revenue, order volume, and fulfillment efficiency across multiple regions and product categories. The dataset was prepared and transformed using Excel and Power BI to support accurate business analysis.

# Data Files

sales_data_raw.xlsx → Original raw dataset (uncleaned)

sales_data_cleaned.xlsx → Processed and structured dataset

# Dashboard Features

KPI Cards: Total Revenue, Total Orders, Completed Orders, Fulfillment Rate (%)

Bar Chart: Revenue by Region

Column Chart: Revenue by Product

Column Chart: Monthly Revenue Trends

Donut Chart: Regional Revenue Distribution

Slicers: Interactive filters for Product and Region

# Business Insights

The North region contributes the highest share of total revenue

Laptops generate maximum sales among all products

Fulfillment performance differs by region, showing operational gaps

Revenue shows seasonal peaks, especially in January and November

# Tools & Technologies

Power BI: Data visualization, KPIs, slicers, dashboard design

Power Query: Data cleaning, transformation, and validation

DAX: Calculated measures for Revenue, Orders, and Fulfillment metrics
